GET /api/tickets
Returns a list of all booked tickets.
[
  {
    "id": 1,
    "passengerName": "John Doe"
  }
]
GET /api/tickets/{id}
Fetch a single ticket by its ID.
id Long — Ticket ID (path variable)
{
  "id": 1,
  "passengerName": "John Doe",
  "source": "Delhi",
  "destination": "Mumbai"
}
POST /api/tickets
Create a new ticket booking.
passengerNameString— Full name
sourceString— Departure city
destinationString— Arrival city
flightDateString— YYYY-MM-DD
seatNumberString— e.g. A12
PUT /api/tickets/{id}
Update passenger details for an existing ticket.
idLong— Ticket ID (path)
bodyJSON— Same fields as POST
{
  "passengerName": "Jane Doe",
  "seatNumber": "B5"
}
DELETE /api/tickets/{id}
Cancel and delete a booking. Returns 200 OK on success with no body.
idLong— Ticket ID to delete

🐳 Dockerfile (Multi-stage Build)
# Stage 1 — Build
FROM maven:3.9.5-eclipse-temurin-17 AS builder
WORKDIR /app
COPY pom.xml .
COPY src ./src
RUN mvn clean package -DskipTests

# Stage 2 — Run
FROM eclipse-temurin:17-jre-alpine
WORKDIR /app
COPY --from=builder /app/target/air-ticket-booking.jar app.jar
EXPOSE 8080
ENTRYPOINT ["java", "-jar", "app.jar"]
